Simple Shortcode to Insert Raw HTML in Hugo

I love the simplicity of creating new content using markdown in Hugo. But sometimes, markdown is not enough – you might want to do a one-off extra-fancy thing with raw html. Surprisingly, Hugo doesn’t seem to have a good way to let you do this in a content file. Here is how to create your own one-line custom shortcode to make that possible. Add a shortcode template to your site, in layouts/shortcodes/rawhtml.
